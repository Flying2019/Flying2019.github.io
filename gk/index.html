颓废计划

名称：高考惊变一百天（暂定）

主要构思

  1. 复现 XJ 高考前 n 天的生活。但不要黑 XJ 太过。
  2. 主要体现一种压抑感。
  3. 策略型游戏，玩家可以控制每个时间段做的事情来最大化收益。
  4. 主打随机性事件，最好有真实感。

界面设计

语言：

暂定用 Unity2D 编写。

好处：

  1. 音效、动画之类的有封装，异步也不用自己写。
  2. 界面友好。
  3. 语言是 C#，看上去和 C++ 很像，入门应该很快。

坏处：

  1. 正式版要钱。
  2. 要熟悉一下面向对象的逻辑。

替代方案：用 Javascript D3.js，还没了解过。

有其他可行的替代方案也可以。

美术：

以像素画为主。

没有可能也可以。

详细设计

游戏目标

在不被踢的情况下在高考中获得尽可能高的分数。

交互内容

有若干事件，诸如完成XX作业、发呆、睡觉等等。

玩家控制一个日程表，可以将事件拖拽进日程表。

在一定范围内玩家可以控制某个事件执行的专注度。

相关系统

能力评价

暂定只有语文和数学两门课。

每门课有多项指标，每项指标分为真实指标和适应度。多数模拟考试评价取决于两者之和，但最终高考（以及个别模拟考）只取决于真实指标。

生理/心理状态

  • 疲劳值：正常睡眠会大量恢复
  • 压力值：正常休息时会缓慢恢复
  • 健康值
  • …

附加属性

  • 心情（暂定）：平静、激动-亢奋、紧张-焦虑、沮丧-抑郁、愤怒
  • 精神问题：失眠（Ⅰ、Ⅱ），困倦，…
  • 健康问题：反胃-腹泻（你知道这是为啥），感冒-发烧
  • …

属性以类似 (de)buff 的形式出现，效果待定。

学习能力

量化为 效率 和 专注度。

效率通过以上各种指标计算得出。

专注度由玩家在一个区间内调整。区间范围取决于以上各种指标（主要取决于压力值和疲劳值）

教师

固定属性

  • 教学水平
  • 佛系程度
  • 易怒值
  • …

相关数值

  • 好感度（名称待修改）：受玩家操作影响
  • 愤怒程度：受特殊事件影响
  • …
